Check/download required libs to run the application

param callback - interface used by Minsitro service to notify the client when the loader is ready param parameters parameters fields:

  • Key Name Key type Explanations "required.modules" StringArray Required modules by your application "application.title" String Application name, used to show more informations to user "qt.provider" String Qt libs provider, currently only "necessitas" is supported. "minimum.ministro.api" Integer Minimum Ministro API level, used to check if Ministro service compatible with your application. Current API Level is 1 ! "minimum.qt.version" Integer Minimim Qt version (e.g. 0x040800, which means Qt 4.8.0, check http://doc.trolltech.com/4.8/qtglobal.html#QT_VERSION)! "3rd.party.repositories" StringArray 3rd party repositories, which should be downloaded by ministro, needs minimum.ministro.api >= 2 Check http://community.kde.org/Necessitas/Ministro for more details.
